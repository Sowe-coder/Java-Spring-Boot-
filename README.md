# Java-Spring-Boot-
 Java Spring Boot backend insurance system with authentication, user data storage, and logging
How to Run

    Clone the repository:

bash

git clone https://github.com/yourusername/java-backend-insurance-system.git
cd java-backend-insurance-system

    Build the project:

bash

./mvnw clean package

    Run the application:

bash

./mvnw spring-boot:run

    Access the application:

        API: http://localhost:8080

        H2 Console: http://localhost:8080/h2-console

        Health Check: http://localhost:8080/actuator/health
